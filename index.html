Abby Tools Suite

A Google Sheets add-on for cross-sheet XLOOKUP, duplicate highlighting, and Google Doc-to-Sheet consolidation.

No coding required. All tools are configured through a sidebar inside Google Sheets.

Features

  • XLOOKUP — match and copy data across sheets or files
  • Highlighter — compare sheets and color matches or unique rows
  • Doc to Sheet — pull rows from a Google Doc table into a Sheet
